# Tetris

Webブラウザで遊べる一人対戦用テトリスゲーム。

## 技術スタック

- HTML5
- CSS3
- JavaScript (ES6+)
- HTML5 Canvas API

## 機能

- 7種類のテトリミノ（I, O, T, S, Z, J, L）
- キーボード操作（移動、回転、ソフトドロップ、ハードドロップ）
- ライン消去とスコアリング
- ポーズ機能

## 操作方法

| キー | 動作 |
|------|------|
| ← | 左移動 |
| → | 右移動 |
| ↓ | ソフトドロップ |
| ↑ | 回転 |
| Space | ハードドロップ |
| P | ポーズ |
